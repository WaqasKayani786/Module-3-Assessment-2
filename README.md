# The Cousins Restaurant

## Module 3 Assessment 2

### Description

The Cousins Restaurant is a responsive online restaurant menu website developed as part of the IT Varsity Module 3 Assessment 2 project. The website allows customers to browse food categories, view menu items, read detailed meal descriptions, select extras, add special requests, and place orders through a simple and user-friendly interface.

---

## Features

* Responsive design for desktop, tablet, and mobile devices
* Homepage displaying food categories
* Meals category page
* Drinks category page
* Desserts category page
* Individual pages for each menu item
* Meal images and descriptions
* Meal prices and serving sizes
* Extras selection using HTML lists
* Special requests input form
* Order confirmation page
* Navigation between all pages
* Custom restaurant branding and design
* Font Awesome icons

## How To Use

1. Open the homepage.
2. Select a food category.
3. Choose a menu item.
4. View the meal details.
5. Select any extras.
6. Add special instructions if required.
7. Click the Place Order button.
8. View the order confirmation page.
